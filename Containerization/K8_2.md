When you run `minikube start`, Minikube launches a **single-node Kubernetes cluster** inside a virtual machine (VM) on your local machine. When you run `minikube ssh`, you're logging **into that VM**. This VM behaves like a small Linux host that runs Kubernetes control plane components and worker node services via Docker containers.
